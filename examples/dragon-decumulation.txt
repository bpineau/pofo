# Dragon-decumulation -- dragon-lite re-tuned for the decumulation years.
#
# STATUS: fully implementable today from an EU/French retail account (four
#   UCITS lines plus one UCITS-eligible ETC), five trades to rebalance.
#   This file is the answer to the honesty note at the bottom of
#   claude-dragonlite.txt, which admits that 60/25/15 "is not the
#   decumulation optimum of its own design space, and it knows it". This
#   is that optimum, kept at five lines. The family now reads: dragon-lite
#   for the accumulation years, hydra-five-engines-ucits as the engineered
#   all-rounder, dragon-decumulation for the years when the flows reverse.
#
# Thesis in one line: same engines as dragon-lite, but tuned for a
#   spending portfolio: a smaller growth core, the trend sleeve split
#   across two genuinely different models, and the freed capital moved
#   into the one asset class the family never held -- inflation-linked
#   bonds, the only instrument whose payoff is written in the retiree's
#   own unit of account.
#
# ## What changes when the flows reverse
#
#   Decumulation is not accumulation run backwards; three asymmetries
#   drive every edit below.
#   1. ORDER STARTS TO MATTER. With withdrawals, two futures with the same
#      average return end differently: selling into a drawdown converts a
#      temporary loss into a permanent one, so the plan's fate is set by
#      its worst starting decade, not its average one (sequence risk:
#      Clare, Seaton, Smith & Thomas, "Reducing Sequence Risk Using Trend
#      Following and the CAPE Ratio", FAJ 2017). Up-capture, dragon-lite's
#      pride, stops paying; drawdown DEPTH and DURATION start costing.
#   2. THE LIABILITY IS EURO INFLATION. An accumulator compounds against
#      no particular yardstick; a euro retiree owes a stream of
#      euro-inflation-indexed spending. Gold and trend hedge inflation
#      PROBABILISTICALLY (they usually respond); an inflation-linked
#      sovereign hedges it BY CONTRACT. A defensive book with no
#      contractual line is a defense built entirely on correlations.
#   3. BLOWUPS CAN NO LONGER BE SAVED AWAY. A working accumulator can
#      out-save a manager failure; a retiree cannot. Concentrations that
#      were tolerable (25% on one trend manager and one method) become
#      structural risks.
#
# ## The four edits vs dragon-lite (what, why, price)
#
#   NTSG 60 -> 45.  The growth engine still pays for the plan, but 60 sat
#     at the aggressive edge of its own sane range and dragon-lite's sweep
#     showed every decumulation metric improving toward 50-55 and below.
#     Price: up-capture and median outcomes (measured below).
#   Trend 25x1 -> 13+13 across two models.  Same total risk budget, single
#     point of failure removed: DBi's replication (DBMFE) and AQR's direct
#     multi-asset trend are different methods with a documented live gap
#     between managers (see hydra's KMLM-vs-AQR note). Price: one more
#     line, and the AQR leg is a fund, not an ETF.
#   Gold 15 -> 15, role sharpened.  Unchanged weight, but in this build it
#     is the stagflation backstop (the regime where trend whipsaws and
#     linkers only tread water) rather than a general diversifier.
#   + 14 linkers (new head).  The freed NTSG points buy euro-area
#     inflation-linked sovereigns: the contractual inflation line, and the
#     lowest-volatility asset in the book, exactly what a withdrawal
#     schedule wants to sell in a bad year. Price: expected return, and it
#     is the one head the backtest cannot vouch for (see the two-lens
#     verdict below).
#
# ## What 100 EUR actually holds (the exposure ledger)
#
#   45 NTSG   -> 40.5 world equity + 27 sovereign futures (~7y duration)
#   13 DBMFE  -> 13   trend (SG CTA replication, DBi)
#   13 AQR MF -> 13   trend (direct multi-asset TSMOM, EUR-hedged share)
#   15 IGLN   -> 15   gold (physical ETC)
#   14 IBCI   -> 14   euro inflation-linked sovereigns
#   ------------------------------------------------------------------
#   Total     ~40.5 equity + ~27 nominal bonds + ~26 trend + ~15 gold
#             + ~14 linkers = ~122.5% (modest embedded leverage, all
#             inside NTSG, no margin account anywhere)
#
# ## Regime map: what hurts a SPENDING portfolio, and which head answers
#
#   Growth boom            40.5 points of equity: presence, deliberately
#                          less of it than any sibling.
#   Disinflation/recession 27 points of intermediate duration + trend
#                          rotating long bonds; linkers hold their value.
#   Deflation crash (2008) Duration + trend + crisis USD. Still the
#                          family's thin quadrant here (see blind spots).
#   Inflation, trending    Trend short bonds / long commodities (Neville,
#                          Draaisma, Funnell, Harvey & van Hemert, JPM
#                          2021) + linkers accruing the index BY CONTRACT.
#   Inflation, choppy      Gold, plus the linkers' accrual. This is the
#   (stagflation)          retiree-killer regime (1966-1981) and the
#                          reason this build exists: dragon-lite meets it
#                          with gold alone.
#   Prolonged equity bear  Trend crisis alpha (Hurst, Ooi & Pedersen, JPM
#                          2017); the linkers are the sleeve you spend
#                          from while equities are underwater.
#
# ## The five heads: rationale, sane ranges, vehicles
#
# HEAD 1 -- NTSG, equity + duration (45%; sane range 35-55%).
#   Global Efficient Core 90/60 (IE00077IIPQ8): 45% of capital carries
#   ~40% world equity AND ~27% intermediate sovereign duration. The range
#   check on this window is remarkably flat across 35-55 (bootstrap ruin
#   4.6-6.6%, stress ruin flat), which is itself the argument: inside the
#   plateau, pick the middle, not the edge (dragon-lite picked the edge
#   and said so). Euro-native variant NTSZ (IE000OV4XWA3) for part of the
#   sleeve trades USD quotation risk for eurozone equity concentration.
#
# HEADS 2a/2b -- trend, two models (13% + 13%; total sane range 20-30%,
#   no single leg above ~15% in this build).
#   Time-series momentum (Moskowitz, Ooi & Pedersen, JFE 2012), the head
#   that de-risks decumulation the most (Clare et al. 2017). The 20%
#   FLOOR is the sweep's strongest empirical result after gold's: at 0%
#   trend, bootstrap ruin nearly doubles (5.5% -> 9.3%) and the stress
#   safe rate drops 20bp; past ~30% the gain flattens while
#   tracking-error regret compounds. Legs: DBMFE (LU2951555403, DBi
#   replication, EUR share) and AQR Managed Futures UCITS (RAEF EUR share
#   LU1662501532; parent LU1103257975), the standout live record among
#   UCITS trend funds. Fallbacks: Winton Trend (IE00BG382Q20), Man AHL
#   Trend (LU0503879685). Both legs at 13 rather than one at 26: the
#   split costs nothing measurable and removes the single-manager risk a
#   retiree cannot earn back.
#
# HEAD 3 -- gold, IGLN (15%; sane range 10-20%).
#   The floor is the single strongest empirical result of the range check:
#   at 0% gold, bootstrap ruin nearly triples (5.5% -> 14.8%) and the
#   stress column degrades sharply; no other head moves the needle that
#   much. The ceiling is a priori: this window worships gold (the sweep
#   happily takes 22%+), but its expected real return is ~0 (Erb &
#   Harvey, "The Golden Dilemma", FAJ 2013) and a spending portfolio
#   cannot lean its median outcome on a zero-carry asset. Vehicle: a
#   physically-backed ETC only (IGLN IE00B4ND3602 at 0.12%, or Amundi
#   GOLD FR0013416716, EUR-quoted). Never an ETN: an ETN is unsecured
#   bank debt, a credit exposure hiding inside the safe-haven line.
#
# HEAD 4 -- euro linkers, IBCI (14%; sane range 10-15%, A PRIORI).
#   iShares Euro Inflation Linked Govt Bond (IE00B0M62X26, 0.09%): euro
#   sovereign linkers, the only line whose cash flows are indexed on the
#   liability itself. Read the honesty paragraph below before quibbling
#   with the weight: this dose comes from liability-matching logic, not
#   from the backtest. Duration caveat, the 2022 lesson: IBCI is
#   all-maturity (~7y duration), and long linkers can lose to rising REAL
#   rates even while inflation runs; the duration-cleaner substitutes are
#   UBS Euro Inflation Linked 1-10 (LU1645380442) for the same euro basis
#   at shorter duration, or UBS TIPS 1-10 EUR-hedged (LU1459801780),
#   which swaps the inflation basis to US CPI in exchange for the deepest
#   available backcast (2000). Any of the three serves the head; IBCI is
#   the most broadly available.
#
# ## The two-lens verdict (read this before believing any number)
#
#   Decumulation portfolios cannot be judged through one lens, because
#   the two available lenses are biased in OPPOSITE directions, and the
#   linkers head sits exactly on the fault line.
#
#   LENS A, the raw common window (2005-11 -> 2026-07 with IBCI; the
#   2000-07 panel with the TIPS proxy for the ruin figures). In-sample,
#   dragon-decumulation vs dragon-lite vs UCITS hydra: CAGR 7.9 / 9.4 / 8.6%,
#   vol 9.9 / 13.0 / 11.0%, Sharpe 0.80 / 0.76 / 0.79, real max drawdown
#   -19.5 / -21.6 / -21.1%, Ulcer 3.8 / 5.3 / 4.6, longest recovery 2.0 /
#   2.9 / 2.7y. Decumulation check (monthly real bootstrap, 40y, rigid 4%
#   real withdrawal, no other income): ruin 5.5 / 9.5 / 2.1%, safe rate
#   at 5% ruin 3.96 / 3.63 / 4.36%. This lens says: best drawdown
#   geometry of the family, worst compounding, and the hydra wins the
#   bootstrap. KNOWN BIAS: resampling 2000-2026 crowns whatever rallied
#   in 2000-2026 (gold, equities). It rewards in-window winners, not
#   forward robustness, and both siblings' files carry the same warning.
#   LENS B, valuation-anchored (the central mean replaced by today's
#   CAPE-implied estimate, risk kept from the fitted weights): safe rate
#   at 5% ruin 2.54 / 2.23 / 2.33%, and the linkers sweep turns
#   monotonically FAVORABLE (0% -> 20% linkers lifts the safe rate 2.38%
#   -> 2.62%). This lens says: if the next decades pay valuation-
#   compressed returns, the lowest-risk book wins and dragon-decumulation leads
#   the family. KNOWN BIAS: it hands every candidate the same forward
#   mean, which flatters low-volatility mixes.
#   WHAT SURVIVES BOTH LENSES: dragon-decumulation is never the worst; it owns
#   the family's best drawdown depth, duration and Ulcer under lens A and
#   its best safe rate under lens B; and dragon-lite reads worst under
#   both decumulation lenses, confirming its own honesty note. What no
#   lens can test: every available linker series starts post-2000, so the
#   1970s, the very regime the head exists for, are OUTSIDE every
#   backcast. The linkers line is bought on the contract, not the chart.
#
# ## Range check honesty (which bounds are measured, which are believed)
#
#   EMPIRICAL (both windows agree): trend floor 20% (ruin nearly doubles
#   below), gold floor ~10% (ruin nearly triples at zero, the strongest
#   single result), NTSG plateau 35-55 (flat; the middle is chosen
#   precisely because the data does not care).
#   A PRIORI (held against in-sample evidence): the gold ceiling (~20%,
#   zero-carry logic vs a window that wants more), the trend ceiling
#   (~30%, behavioral, the backtest would take more of the shared-backcast

#   lens A reads against it for the stated in-sample reasons). If you
#   need every weight blessed by a backtest, hold the hydra; this file
#   spends 14 points on a contract the chart cannot price.

# ## Blind spots (known, accepted, and the patch for each)
#
#   (a) DEFLATION CONVEXITY, still the thin quadrant: 27 points of
#       intermediate futures, no long end. The patch is 5-8% IDTL/DTLE
#       funded from NTSG, at which point you are converging toward a
#       hydra-with-linkers; accepted here to hold the five-line budget.
#   (b) LINKERS BASIS: IBCI carries ~7y duration (real-rate risk, 2022);
#       the 1-10 substitutes above trade that against basis (US CPI) or
#       availability. Pick one deliberately; do not hold none.
#   (c) TREND WINTERS: 26 points across two managers still spends years
#       flat while equities run (2011-2019). The split removes blowup
#       risk, not boredom risk; the cure is written rules, not weights.
#   (d) CURRENCY: NTSG and IGLN are USD-quoted and unhedged (half
#       feature: crisis USD); the AQR leg is EUR-hedged, IBCI and the
#       withdrawal liability are euro-native. Residual USD exposure is
#       roughly half the book, deliberate but real.
#   (e) RECONSTRUCTION RISK: deep trend history shares one TSMOM backcast
#       (understates manager dispersion); linkers exist only post-2000;
#       gold pre-1975 is a regulated price. Regime-behavior study, not a
#       promise.
#   (f) NO CASH BUFFER IN THE FILE: this is the invested book only. Pair
#       it with 2-3 years of spending in euro liquidity (livrets, XEON,
#       fonds euros; see fire-liability-buffer.txt), held OUTSIDE these
#       weights, and spend from the buffer in drawdown years.
