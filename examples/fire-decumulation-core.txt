# FIRE Decumulation Core (UCITS, curated) -- the cruise allocation
# Idea: a ~3% real-withdrawal portfolio for a European early retiree. One
#   capital-efficient engine (NTSG: 90% world equity + 60% bond futures),
#   a trend sleeve split over TWO distinct engines (CTA replication + direct
#   trend), gold, and a defensive pocket that actually matches the liability:
#   short euro inflation-linked bonds plus EUR cash. Four return engines,
#   ~130% look-through notional, targeting ~10% portfolio volatility.
# Inspired by: Miller 2017 (trend raises safe withdrawal rates); Neville,
#   Harvey et al. 2021 (trend + gold + short linkers in inflationary times);
#   Anarkulova et al. 2025 (global data say withdraw ~2.3-3%, not 4%);
#   Return Stacked / WisdomTree efficient core.
# The withdrawal below is 3%/yr of the start capital; sequence risk is the
#   enemy, so compare with fire-bond-tent-departure (the year-0 variant).
