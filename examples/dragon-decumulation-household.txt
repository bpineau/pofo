# Dragon-decumulation, household edition -- the whole-wealth build, wrappers included.
#
# STATUS: fully implementable today from a French retail setup. This is
#   dragon-decumulation.txt expressed as a real household holds it: across a CTO
#   AND the French tax shelters (PEA / PEA-PME), which by law can hold
#   nothing but European-domiciled equity funds. The shelters are not a
#   footnote; they reshape the build, and the reshaping turns out to be a
#   measured upgrade, not a compromise. Nine lines, two sections below.
#
# Thesis in one line: dragon-decumulation's design (a smaller efficient core,
#   trend split across two managers, gold, a contractual euro-inflation
#   line), with the equity sleeve carved out of the 90/60 core and moved
#   into the tax shelters as plain world equity plus a small-cap
#   value/quality tilt, and the bond overlay lost in the carve-out
#   rebuilt with a dedicated long-duration line.
#
# ## Lineage: Browne -> Cole -> dragon-lite/hydra -> dragon-decumulation -> here
#
#   The design philosophy is regime balancing, four generations on.
#   Harry Browne's Permanent Portfolio (1981) planted the idea: hold one
#   winner per economic season (growth, recession, inflation, deflation)
#   and let rebalancing harvest the rotations, instead of forecasting.
#   Chris Cole's Dragon (Artemis, 2020) rebuilt it for century-scale
#   cycles: add trend and long volatility, size for survival. The family
#   in this directory made it buyable: dragon-lite distilled it to three
#   UCITS lines around a capital-efficient 90/60 core (Asness, "Why Not
#   100% Equities", JPM 1996), the hydra engineered it into one head per
#   return engine, and dragon-decumulation re-tuned it for decumulation, where
#   sequence risk and a euro-inflation liability change what "optimal"
#   means (see that file for the full argument).
#   This edition adds the last real-world layer: TAXES AND WRAPPERS. A
#   French household keeps part of its wealth in a PEA and a PEA-PME
#   (gains taxed ~17.2% instead of ~30%), and those wrappers accept only
#   European equity funds: no trend, no gold, no bonds, no linkers ever.
#   The naive reading is a constraint; the correct reading is a division
#   of labor. The shelters take the equity sleeve (they can hold nothing
#   else, and equity is the sleeve with the largest expected gains, i.e.
#   the most tax to shelter); the CTO takes everything the shelters
#   cannot hold. Carving the equity out of NTSG then forces two edits
#   that the hydra had already argued for on pure design grounds: a
#   small-cap value tilt (the shelters' best funds are exactly that) and
#   a dedicated long-duration line (to rebuild the bond overlay the
#   carve-out removed). The constraint re-derives the engineering.
#
# ## What 100 EUR actually holds (the exposure ledger)
#
#   CTO section
#   28 NTSG   -> 25.2 world equity + 16.8 sovereign futures (~7y)
#    4 MTH    ->  4   25+y eurozone sovereigns (~20y duration,
#                     ~11 points in 7y-equivalent terms)
#   13 DBMFE  -> 13   trend (SG CTA replication, DBi)
#   13 AQR MF -> 13   trend (direct multi-asset TSMOM, EUR-hedged)
#   15 IGLN   -> 15   gold (physical ETC)
#   13 IBCI   -> 13   euro inflation-linked sovereigns
#   PEA / PEA-PME section (equity by law)
#    5 WPEA   ->  5   world equity (synthetic swap inside the PEA)
#    7 Indep. Europe Small  -> 7  European small value/quality
#    2 Indep. France Small Mid -> 2  French small/mid value/quality
#   ------------------------------------------------------------------
#   Total: ~39 equity (of which 9 small value) + ~26.5 nominal duration
#   (7y-equivalent) + 26 trend + 15 gold + 13 linkers  = ~119.5%
#   exposure, all embedded leverage inside NTSG, no margin anywhere.
#   The ledger lands on dragon-decumulation's targets almost exactly; only the
#   internal composition of the equity sleeve differs (a 9-point
#   small-value tilt instead of pure cap-weight).
#
# ## Regime map: what hurts a spending household, and which line answers
#
#   Growth boom            ~39 points of equity: presence. The small
#                          value tilt diversifies the mega-cap-growth
#                          concentration a world index carries.
#   Disinflation/recession NTSG's futures + MTH + trend rotating long
#                          bonds; linkers hold their value.
#   Deflation crash (2008) MTH is the line bought for this: the long
#                          end is what actually rallies in a flight to
#                          quality, here in the liability currency.
#                          Caveat: the small-value tilt DEEPENS this
#                          regime (small caps fell harder in 2008).
#   Inflation, trending    Trend short bonds / long commodities (Neville,
#                          Draaisma, Funnell, Harvey & van Hemert, JPM
#                          2021) + linkers accruing the index by contract.
#   Inflation, choppy      Gold + the linkers' accrual; historically
#   (stagflation)          value tilts also weathered the 1970s better
#                          than growth. The retiree-killer regime, and
#                          the reason the linkers line exists.
#   Prolonged equity bear  Trend crisis alpha (Hurst, Ooi & Pedersen,
#                          JPM 2017); linkers and MTH are what you
#                          spend from while equities are underwater.
#
# ## Why the wrappers change behavior, not just taxes
#
#   Three consequences worth spelling out, because none shows up in a
#   backtest. SPENDING ORDER: the shelters are the cheap spending pocket
#   (~17.2% on gains vs ~30%), so a decumulating household drains them
#   early; since they hold only equity, spending them first de-equitizes
#   the early years and lets the equity share drift back up later. That
#   is an implicit rising-equity glidepath (Kitces & Pfau, 2014), the
#   sequence-friendly shape, obtained for free from tax logic.
#   REBALANCING ASYMMETRY: money cannot move freely INTO the shelters
#   (deposit ceilings) and cannot change asset class inside them, so all
#   cross-sleeve rebalancing happens in the CTO section; inside the PEA
#   the only trade is equity-for-equity. Set bands accordingly.
#   CONTRIBUTION CEILINGS: the PEA block's weight is partly set by law,
#   not by choice; treat its range as descriptive, and let the CTO
#   absorb the drift.
#
# ## The lines: role, sane range, equivalent-quality alternatives
#
# CTO -- NTSG, equity + duration core (28%; sane range 24-35% HERE).
#   Global Efficient Core 90/60 (IE00077IIPQ8). Its weight in this
#   edition is a COMPLETION: NTSG's equity plus the PEA block must land
#   household equity near 35-45%, so NTSG shrinks when the shelters
#   grow. Euro-native variant NTSZ (IE000OV4XWA3) for part of the
#   sleeve trades USD quotation for eurozone concentration.
#
# CTO -- MTH, deflation duration (4%; sane range 3-6%).
#   Amundi Euro Government Bond 25+Y Acc (LU1686832194, 0.07%): the
#   cheapest and largest of the eurozone 25+ ETFs, euro-native long
#   duration, accumulating. Pure arithmetic first: every euro of equity
#   carved out of NTSG lost 0.60 EUR of ~7y bond overlay; at ~20y
#   duration this line rebuilds it at about 2.8x, so ~4 points restore
#   what ~11 points of overlay would have carried. It also patches the
#   dragon family's thinnest quadrant (2008-style convexity), in the
#   liability currency and with no coupons to declare.
#   WHY THIS LINE, from a survey of the whole category rather than the
#   toolkit's habits. Every EUR-hedged long-Treasury UCITS is
#   DISTRIBUTING (DTLE IE00BD8PGZ49; Amundi Long Dated LU1407890976
#   likewise) and the only accumulating Treasury shares are unhedged
#   (DTLA IE00BFM6TC58, IDTL); a Bund-only LONG ETF does not exist (the
#   Germany funds are all-maturity, ~8y), so eurozone-mix fragmentation
#   risk is the unavoidable price of euro long duration; and among the
#   eurozone 25+ funds this one dominates on fees and size (0.07% and
#   ~350m vs 0.15% and ~110m for Xtrackers DBXG LU0290357846, the
#   nearest equivalent). Remaining prices, stated: a market-weighted
#   sovereign mix (heavy France/Italy), so a 2011-style euro sovereign
#   crisis would wobble the hedge exactly when equity also suffers.
#   Equal-quality alternatives: DBXG (same exposure), Amundi 15+Y Acc
#   (LU1287023268, milder duration), Xtrackers Global Government Bond
#   EUR-hedged 1C (LU0378818131, ~8y but issuer-diversified and 700m+),
#   DTLE if you accept coupons for the US issuer, DTLA/IDTL for Acc
#   plus the crisis-USD kicker. On the measured panels all of these,
#   including dropping the line for +2 linkers +2 gold, land within
#   noise of each other: the line is bought on qualitative grounds;
#   hold whichever variant you will not fiddle with.
#
# CTO -- trend, two models (13% + 13%; total sane range 20-30%).
#   Same head, same reasoning, same vehicles as dragon-decumulation: DBMFE
#   (LU2951555403, DBi replication) + AQR Managed Futures UCITS
#   (RAEF EUR LU1662501532, direct TSMOM; fallbacks Winton IE00BG382Q20,
#   Man AHL Trend LU0503879685; for 100k+ tickets the same fund's flat I
#   class IAE2F LU1662499612 at OCF ~0.88% vs 1.31% saves ~0.43 pts/yr,
#   minimum and KID eligibility to verify). The 20% floor is empirical (ruin nearly
#   doubles below); the split removes single-manager risk a spending
#   household cannot earn back; the ~30% ceiling is behavioral.
#
# CTO -- gold, IGLN (15%; sane range 10-20%).
#   Physically-backed ETC only (IE00B4ND3602, 0.12%; equal quality:
#   Amundi GOLD FR0013416716, EUR-quoted, or Invesco SGLD). Never an
#   ETN (unsecured bank debt inside the safe-haven line). The ~10% floor
#   is the strongest empirical result of the family's range checks (at
#   0% gold, modeled ruin roughly triples); the ceiling is a priori
#   (zero expected real carry: Erb & Harvey, FAJ 2013).
#
# CTO -- euro linkers, IBCI (13%; sane range 10-15%, A PRIORI).
#   iShares Euro Inflation Linked Govt (IE00B0M62X26, 0.09%): the only
#   line whose cash flows are indexed on the household's own liability.
#   Duration caveat (the 2022 lesson): IBCI is all-maturity (~7y), and
#   long linkers can lose to rising REAL rates while inflation runs;
#   duration-cleaner substitutes are UBS Euro Inflation Linked 1-10
#   (LU1645380442) or UBS TIPS 1-10 EUR-hedged (LU1459801780, US-CPI
#   basis, deepest backcast). DEEP-WINDOW TIP: IBCI's 2005 quotes are
#   what bounds this file's panel; to visualise the build through the
#   2000-02 crash (report curves or the FIRE explorer), temporarily
#   swap this line for LU1459801780, whose backcast reaches 2000-06.
#   Same role, known biases: US-CPI-hedged basis and milder duration
#   (vol 3.9% vs 5.7%, CAGR 1.70% vs 1.99% on 2005-2026, monthly corr
#   ~0.6, so it understates IBCI's real-rate risk). Visualisation
#   stand-in only: the held line stays IBCI, the liability basis.
#   Dose bought on the contract, not the
#   chart: see the two-lens verdict.
#
# PEA -- WPEA, world equity (5%; sane range 3-8%).
#   iShares MSCI World Swap PEA (IE0002XZSHO1, 0.20%): full world
#   exposure inside a wrapper that legally holds only European funds,
#   via a synthetic swap (the fund holds EU equities and swaps their
#   return for the index's; collateralized and reset, a small
#   counterparty exposure listed in the blind spots). Equal quality:
#   Amundi CW8 (LU1681043599, older and a touch dearer) or EWLD. One
#   world line per PEA is plenty; never hold cash-like assets here.
#
# PEA + PEA-PME -- Independance AM Europe Small (7%, held across both
#   wrappers) and France Small Mid (2%); small sleeve sane range 8-15%.
#   The equity tilt: European small-cap value/quality from a boutique
#   with a three-decade record and a disciplined "value pere de famille"
#   process. Design-wise this IS the hydra's small-value head (size and
#   value premia: Fama & French 1992/93; junk-screened smalls: Asness,
#   Frazzini, Israel, Moskowitz & Pedersen, JFE 2018), with two
#   advantages over the hydra's ZPRV: it is European (diversifying an
#   otherwise US-heavy book) and it is shelter-eligible (the premium
#   compounds at 17.2% tax, not 30%). The France Small Mid line is a
#   legacy overlap; folding it into Europe Small over time is fine.
#   Above ~15% the sleeve stops diversifying and starts doubling down
#   on equity beta. CTO-only alternatives: ZPRV (IE00BSPLC413), AVWS
#   (IE0003R87OG3). Second-manager options inside a PEA-PME if the
#   sleeve grows: Gay-Lussac Microcaps, HMG Decouvertes.
#
# ## The two-lens verdict (and what was actually measured)
#
#   Decumulation builds cannot be judged through one lens; the two
#   available lenses are biased in opposite directions (the full
#   argument lives in dragon-decumulation.txt). Measured on the toolkit's
#   deep panels, with two declared proxies for the young lines (ZPRV
#   stands in for the Independance sleeve, the TIPS-hEUR backcast for
#   IBCI; window 2000/2005 -> 2026):
#   LENS A, raw window (monthly real bootstrap, 40y, rigid 4% real
#   withdrawal): ruin 3.1% vs 5.5% for pure dragon-decumulation and 2.1% for
#   the hydra; safe rate at 5% ruin 4.17% vs 3.96% and 4.36%. The
#   small-value and duration lines close most of dragon-decumulation's
#   in-window gap to the hydra. Bias: resampling 2000-2026 crowns that
#   window's winners.
#   LENS B, valuation-anchored (central mean replaced by the CAPE-
#   implied estimate, risk kept from the fitted weights): safe rate
#   2.52% vs 2.54% (dragon-decumulation) and 2.33% (hydra). The low-risk book
#   keeps the family lead. Bias: equal forward mean flatters low vol.
#   WHAT SURVIVES BOTH: this edition is the family's best-of-both, at
#   or near the top under each lens while pure dragon-decumulation wins only
#   lens B and the hydra only lens A. What no lens can test: every
#   linker series starts post-2000, so the 1970s, the regime the
#   linkers exist for, sit outside every backcast. Every non-linker
#   line now carries a sim:on backcast (WPEA to 1971, MTH to 1970, the
#   Independance sleeve to 1990 via its France sibling), so the file's
#   panel reaches IBCI's 2005 start, the deepest any euro-linker book
#   can go. Mind the window when comparing with siblings fitted on
#   1988+ panels: same plan, different window, different verdict is a
#   simulator artifact, not a portfolio property; on an identical
#   panel this build matches or beats them (the tables above; to see it
#   yourself, window the siblings with the CLI flag, e.g.
#   `pofo -start 2000-07-01 -fire examples/claude-dragonlite.txt`).
#
# ## Range check honesty (measured floors, believed ceilings)
#
#   EMPIRICAL: trend floor 20% and gold floor ~10% (the family's two
#   strongest sweep results, stable across windows); small sleeve 8-15%
#   (the hydra's sweep, and dragon-lite's own file measured that moving
#   12 points of core into small value was its single best edit); NTSG
#   plateau (flat across 24-35 here: pick the middle).
#   ARITHMETIC: MTH 3-6% (the duration-completion computation above).
#   A PRIORI: the linkers dose 10-15% (liability matching; lens B
#   supports it, lens A reads against it for stated in-sample reasons),
#   the gold ceiling (~20%, zero-carry logic against a window that
#   worships gold), the trend ceiling (~30%, behavioral).
#
# ## Blind spots (known, accepted, and the patch for each)
#
#   (a) MANAGER CONCENTRATION: ~9 points on one small boutique across
#       two wrappers. Accepted at this weight for the record and the
#       shelter eligibility; the patch is a second PEA-PME manager if
#       the sleeve ever exceeds its range.
#   (b) SMALL-CAP CYCLICALITY: the tilt deepens 2008-style liquidity
#       crashes and can trail cap-weight for years (2015-2020). It is
#       equity beta with a premium, not a hedge; MTH and trend exist
#       for the regime it worsens.
#   (c) SWAP COUNTERPARTY inside WPEA/CW8: collateralized, reset, and
#       small, but real; the price of world exposure inside a wrapper
#       that bans it. Holding part of the world sleeve as physical ETFs
#       on the CTO (via NTSG) already dilutes it.
#   (d) DEFLATION CONVEXITY: improved by MTH but still the modest
#       quadrant; the family accepts this rather than pay negative
#       carry for long-vol convexity (see dragon-decumulation and the hydra).
#   (e) TREND WINTERS: 26 points can spend years flat while equities
#       run; the cure is written rules, not weights.
#   (f) CURRENCY: roughly half the book is USD-quoted unhedged (NTSG,
#       IGLN), half feature (crisis USD), half bug; the AQR leg is
#       EUR-hedged, MTH, IBCI and the liability are euro-native.
#   (g) RECONSTRUCTION AND WINDOW LIMITS: shared TSMOM backcast for
#       deep trend history; every non-linker line is backcast far past
#       its listing (WPEA to 1971, MTH to 1970, the Independance
#       sleeve to 1990 via its France sibling), so the panel binder is
#       the linkers line: 2005 with IBCI, 2000-06 with its LU1459801780
#       stand-in (see the linkers head), and nothing earlier anywhere,
#       because inflation-linked markets did not exist (first linkers:
#       UK 1981, TIPS 1997, OATi 1998, euro ~2003).
#       Regime-behavior study, not a promise.
#   (h) NO BUFFER IN THE FILE: pair with 2-3 years of spending in euro
#       liquidity held outside these weights (livrets, fonds euros,
#       XEON; see fire-liability-buffer.txt), spent in drawdown years.
#
# ## Honesty note on the WEIGHTS
#   28/5/7/2/4/13/13/15/13 are hand-rounded from dragon-decumulation's targets
#   minus the wrapper carve-out arithmetic; nothing was optimizer-fit.
#   The two-lens table is the full disclosure; expect measured edges to
#   shrink out of sample. Anywhere inside the sane ranges is the same
#   portfolio. The design carried across editions, a contractual
#   inflation line plus split trend around a smaller efficient core,
#   with the equity sleeve now living where the tax code pays for it,
#   is the part built to survive.
#
# ## Practical notes
#   - Shelter hygiene: one world line per PEA (consolidate duplicates),
#     no cash parked inside, and never bend the shelters toward assets
#     they cannot hold; the CTO exists for that.
#   - Spending order: drain the shelters first in decumulation (17.2%
#     vs 30% on gains); the implicit glidepath above is a feature.
#   - Rebalance quarterly or on bands, executed in the CTO section;
#     inside the shelters only equity-for-equity trades exist.
#   - The AQR leg is a FUND, not an ETF: check platform availability
#     (see dragon-decumulation.txt for the fallback ladder).
#   - Pair with a floor-capped flexible withdrawal rule, not a rigid
#     indexed one; the portfolio is half the machine.
