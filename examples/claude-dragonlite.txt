# Claude dragon-lite -- the Dragon Portfolio distilled to three buyable lines.
#
# STATUS: fully implementable TODAY with UCITS/ETC vehicles, no compromise
#   and no waiting for product launches: NTSG is UCITS, DBi has a UCITS EUR
#   share (DBMFE), gold is a physical ETC. This is the simplicity flagship
#   of the family: the portfolio you can explain in one sentence, rebalance
#   in three trades, and hold through a decade without second-guessing six
#   sleeves. Its engineered sibling is hydra-five-engines.txt.
#
# ## Lineage: Cole's Dragon, minus the hardest head, plus a growth engine
#   The ancestor is the Artemis Dragon Portfolio (Christopher Cole, "The
#   Allegory of the Hawk and Serpent", Artemis Capital 2020; see
#   dragon-portfolio-artemis.txt): ~24% equity, 18% long bonds, 19% gold,
#   18% commodity trend, 21% long volatility, built to compound through
#   100-year regime cycles rather than one favorable decade. Dragon-LITE
#   makes two deliberate departures:
#   - It DROPS the long-volatility head. Long vol is the Dragon's hardest
#     sleeve to hold in practice: persistent negative carry, enormous
#     manager dispersion, and no cheap UCITS vehicle that reliably delivers
#     the convexity (the investable funds mostly bleed). Trend + gold +
#     duration take over the crisis job, imperfectly: they are
#     PROBABILISTIC hedges (they usually respond), not the contractual
#     convexity Cole wanted. That is the "lite".
#   - It RE-WEIGHTS toward growth. Cole's 24% equity is calibrated for
#     century-scale survival; a FIRE accumulator/decumulator still needs
#     the portfolio to compound. Via NTSG the equity share rises to ~54%,
#     with the bond sleeve carried as futures ON TOP (capital efficiency:
#     Asness, "Why Not 100% Equities", JPM 1996; WisdomTree Efficient
#     Core), instead of Cole's side-by-side 24+18.
#
# ## What 100 EUR actually holds (the exposure ledger)
#
#   60 NTSG   -> 54  world equity + 36 sovereign futures (~7y duration,
#                    US/Germany/UK/Japan)
#   25 DBMF   -> 25  managed-futures trend (SG CTA replication, DBi)
#   15 XAU    -> 15  gold
#   ------------------------------------------------------------------
#   Total     ~54 equity + ~36 bonds + ~25 trend + ~15 gold  = ~130%
#
# ## Regime map: what hurts, and which head answers
#
#   Growth boom            54 points of world equity: full presence, and
#                          the best up-capture of the family (61%).
#   Disinflation/recession 36 points of intermediate duration + trend
#                          rotating long bonds.
#   Deflation crash (2008) The KNOWN GAP: only intermediate duration (no
#                          long-end convexity, no long vol). Trend and the
#                          USD flight-to-quality carry the day; measured
#                          2008 real drawdown -18% vs -13% for the hydra,
#                          which holds a deep-duration head for this.
#   Inflation shock        Trend long commodities/short bonds (Neville,
#                          Draaisma, Funnell, Harvey & van Hemert, "The
#                          Best Strategies for Inflationary Times", JPM
#                          2021) + gold.
#   Choppy stagflation     Gold only; trend whipsaws without a trend
#                          (Neville et al. 2021).
#   Prolonged equity bear  Trend crisis alpha (Hurst, Ooi & Pedersen, "A
#                          Century of Evidence on Trend-Following
#                          Investing", JPM 2017; for the decumulation
#                          link, Clare, Seaton, Smith & Thomas, FAJ 2017).
#
# ## The three heads: rationale, sane ranges, vehicles
#
# HEAD 1 -- NTSG, equity + duration (60%; sane range 50-65%).
#   90% developed-world equities + 60% sovereign futures in one UCITS line
#   (IE00077IIPQ8). The 60% weight is the IDENTITY choice of this build:
#   growth-first, hedges second. Empirically the ceiling is hard: above
#   ~65% every decumulation metric degrades fast (worst rolling 5y CAGR
#   falls to -2.5%, ruin at a 4% withdrawal doubles). The floor is
#   identity, not data: BELOW ~50% the portfolio stops being dragon-lite
#   and converges toward the hydra; and honesty demands saying the sweep
#   PREFERS that direction (see the honesty note). Approaching
#   decumulation, slide toward the 50-55% end. Euro-native variant: NTSZ
#   (IE000OV4XWA3) for part of the sleeve kills the FX quotation on it.
#
# HEAD 2 -- trend / managed futures (25%; sane range 20-30%).
#   Time-series momentum (Moskowitz, Ooi & Pedersen, JFE 2012), the one
#   head that can profit in both tails and the main reason this build
#   de-risks decumulation at all. The 20% floor is the strongest
#   empirical result of the sweep: at 0-15% trend, ruin runs 1.2-1.5%,
#   max drawdown -32 to -39%, recoveries beyond 5 years. The 30% ceiling
#   is behavioral (tracking-error regret) plus single-model prudence, not
#   backtest (which happily takes 40%: in-sample flattery of the shared
#   TSMOM backcast). KNOWN WEAKNESS: 25% on ONE manager and ONE method
#   (DBi replication). The single best upgrade to this file is splitting
#   it two ways, e.g. 13% DBMFE (LU2951555585, EUR share) + 12% AQR
#   Managed Futures UCITS (LU1103257975, the standout live record
#   2015-2026) -- at which point you are two thirds of the way to the
#   hydra's trend sleeve. Rejected for the base build: WINTON-TREND-
#   EQUITY (portable alpha, adds ~8 points of hidden equity), MFA8 (same
#   DBi method as DBMFE, ESG-amputated commodities: not diversifying).
#
# HEAD 3 -- gold (15%; sane range 10-20%).
#   Near-zero expected real return alone (Erb & Harvey, "The Golden
#   Dilemma", FAJ 2013), so the ceiling is drag (~0.17 pt of CAGR per 5
#   points of gold in the backtest); the floor is empirical (at 0% gold,
#   ruin nearly doubles and the worst-5y deepens: it is the only backstop
#   in choppy stagflation). Cole's Dragon runs 19%; 15% is the same idea
#   net of the growth tilt. Vehicle: a physically-backed ETC (IGLN
#   IE00B4ND3602, Invesco SGLD, or Xetra-Gold for a EUR line).
#
# ## Blind spots (known and accepted; the patches are one file away)
#
#   (a) DEFLATION CONVEXITY. No long-duration head and no long vol: in a
#       2008-style crash this build leans on intermediate futures, trend
#       reacting in time, and the USD. It worked (2008 real drawdown -18%)
#       but it is the family's widest gap; 3-5% of IDTL/DTLE or accepting
#       the hydra's ZROZ-style head is the patch.
#   (b) SEQUENCE RISK, measured: worst rolling 5y CAGR is NEGATIVE
#       (-1.7%) and the worst recovery runs 4.9 years (vs +0.9% and 3.2y
#       for the UCITS hydra). A retirement starting into a bad decade CAN
#       spend five-plus years under water. This is the price of the 54%
#       growth engine; see the honesty note.
#   (c) SINGLE-MANAGER TREND: 25% on one fund and one replication method.
#       Upgrade path in HEAD 2.
#   (d) Currency: all three lines are USD-quoted, unhedged; EUR/USD vol
#       partly feeds portfolio vol. Half feature (crisis USD), half bug.
#       Uncomment "#meta currencies:USD,EUR" below to SEE the gap: one
#       column per currency, each with its own numeraire and inflation.
#   (e) Euro-area inflation is hedged only indirectly (gold, trend). The
#       liability of a EUR retiree is euro inflation: approaching
#       decumulation, add 7-15% of euro liquidity (livrets + XEON/ERNX +
#       short euro linkers; the commented CHSN/42C0 lines below are that
#       buffer, kept at hand).
#   (f) A Japan-style lost decade: trend and duration soften, nothing
#       profits. Optional patch: a small discretionary-macro head (the
#       commented BHMG line; see fire-macro-leg-bhmg.txt).
#   (g) Reconstruction risk: pre-inception history is backcast (SIM
#       series); trend backcasts understate live manager dispersion.
#       Regime-behavior study, not a promise.
#
# ## Honesty note on the WEIGHTS
#   60/25/15 are round, hand-picked numbers: a priori, never optimizer-
#   tuned, which is worth something out of sample. The RANGE CHECK
#   (per-head sweep, rest renormalized, 1988-2026 EUR + bootstrap ruin at
#   a 4% real withdrawal over 40y) confirms the trend floor (20%) and the
#   gold floor (10%) emphatically, and the NTSG ceiling (~65%) clearly.
#   But it also says something uncomfortable: the current point sits at
#   the AGGRESSIVE EDGE of its own sane ranges. Every decumulation metric
#   improves monotonically toward 50-55% NTSG / ~30% trend (ruin 0.77% ->
#   ~0.5%, worst-5y -1.7% -> -0.9%), and the improvement direction, taken
#   further, simply becomes the hydra. Dragon-lite at 60/25/15 is
#   therefore an ACCUMULATION-leaning tuning of the concept, chosen for
#   up-capture and simplicity; it is not the decumulation optimum of its
#   own design space, and it knows it.
#
