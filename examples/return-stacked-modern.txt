# Return Stacking -- stocks, bonds, trend and gold, all capital-efficient
# Idea: stack uncorrelated return streams so each euro works more than once.
#   RSSB gives 100% global equity + 100% US Treasuries per euro; RSST adds a
#   100% managed-futures (trend) overlay on 100% US equity; GDE layers 90%
#   gold on 90% US equity. Together about 200% notional on 100% capital:
#   roughly 100% equity, 60% bonds, 25% trend and 14% gold in one package
#   that never sells the equity down to make room for the diversifiers.
# Inspired by: Return Stacked ETFs (Newfound/ReSolve, Corey Hoffstein) for
#   RSSB/RSST; GDE is WisdomTree's take on the same capital-efficiency idea.
#   https://www.returnstackedetfs.com/
# US-listed (no UCITS equivalent yet); history extended via simulation.
#meta rebalance:90
#meta sim:on

25    RSST        # Return Stacked US Stocks & Managed Futures: 100% US equity + 100% trend
15    GDE         # WisdomTree Efficient Gold Plus Equity: 90% US equity + 90% gold (90/90)
